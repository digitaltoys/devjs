# Flash card app
단어 암기 앱을 만들어봅시다

> # Apple

## 1. 단계 
- 단어와 뜻을 담은 object를 만듭니다. 
- 단어만 화면에 출력합니다
- 단어를 클릭하면 뜻을 보여주고, 다시 뜻을 클릭하면 단어를 보여줍니다
